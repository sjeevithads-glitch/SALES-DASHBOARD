# Power BI Dashboards

## 📊 Overview
I recently completed my learning journey in **Power BI** and explored its features by creating multiple interactive dashboards. These dashboards transform raw data into meaningful insights, helping management and stakeholders make informed decisions.

## 🚀 Dashboards Created
- **Hospital Analytics Dashboard**  
  KPIs on admissions, insurance claims, and readmission rates. Delivered business insights that guided management decisions and earned appreciation for highlighting key trends.  

- **Employee Attrition Dashboard**  
  Analyzed workforce trends using DAX measures and data modeling.  

- **Sales Performance Dashboard**  
  Visualized profit, sales growth, and regional contributions with interactive charts.
